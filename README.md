## 📌 Overview

This POS system is designed to simulate a **real-world retail environment**, including sales, inventory, customers, reporting, promotions, and secure authentication.

It is optimized for **practical business usage in Lebanon**, where dual-currency handling is essential.

---

## 🚀 Core Features

### 🛍️ Sales & Checkout System
- Barcode scanning & product search
- Fast cart-based checkout
- Real-time price calculation

### 💵 Multi-Currency Payments
- LBP / USD support
- Cash, card, and mixed payments
- Automatic conversion handling

### 📦 Inventory Management
- Stock tracking system
- Batch & expiry date control
- Low-stock alerts

### 🎯 Promotions Engine
- Percentage discounts
- Fixed discounts
- Buy X Get Y (BOGO)

### 🎁 Loyalty System
- Customer points accumulation
- Reward/stamp system

### 📊 Advanced Reporting
- Sales analytics
- Profit tracking
- Cashier performance reports
- Export to Excel & PDF

### 🔐 Security & Roles
- Admin / Cashier roles
- Secure authentication system
- Password hashing (BCrypt)

### 💾 Data Management
- Backup & restore system
- SQLite lightweight database
